dat

Share, version, and sync data.

Dat syncs data across the distributed web. Optimized for speed, simplicity, and security.
View on GitHub
Try Dat
Dat works on the web, with the Dat.js client library, in the Dat Desktop application, from the Beaker Web Browser, or from the command line:
1
Install Dat: 
npm install -g dat
2
Share a directory by typing: 
dat path/to/directory
3
And that’s it! Anyone you share the link with can now access the data.
Distributed Sync
Dat syncs and streams data directly between devices, putting you in control of where your data goes.
Efficient Storage
Data is deduplicated between versions, reducing bandwidth costs and improving speed.
Data Preservation
Dat uses Secure Registers to ensure data is trusted, archived, and preserved.
